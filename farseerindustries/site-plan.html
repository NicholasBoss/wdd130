Overview

Purpose

Ever want to display all you hard work in one place for employers to see? My website will display all the accomplishments I have done in my time here at BYU-Idaho, and it can be applicable to any who want to display their own work on their own website. This website will become a template for any to use with little to no instructions on how to build it. Using this website will reduce the amount of paper you use and you can bring a mobile device (preferrably a tablet) to any interview to show off your accomplishments.

Audience

This site is focused toward any who want to make an aesthetically pleasing site to show off their accomplishments. Whether it is for a school project or a business interview, this site is just for you. This is also for employers who are looking for ways to suggest a helpful format they may want to see in an interview. The site can be used as is (and updated accordingly) for each individual who wants to store their accomplishments for anyone to see.

Content

Home page

My site will present all the project and programs I have created so employers can easily see what I have done and they can see what I could contribute to their company. I hope that this site will help others to understand what I am capable of and to see examples of programs I could build or contribute to building. People will visit my site since it will be the only place anyone can see my accomplishments in an organized fashion. I hope to use this site to add more accomplishments as I continue to make more programs.

HOME PAGE:

  • Headers stick to the top of the page (position: fixed) allowing easy transition between the different pages on my site. Header will contain links to: HOME, ABOUT ME, PROJECTS, and HOBBIES pages.
  • Footer will contain links to: GITHUB, SITE PLAN, and ABOUT ME page
  • Boxes that represent each project I have completed or am working on with an image from the project. (three per line) with box shadow and hover interaction. Boxes will act as the link to each project.
  • Underneath the boxes will have a short description of the project, like the name.
  • Most recent project page will be updated continuously. Link in header will be the same as the link on the box to that project. This project page will be designed like the other project pages.
  • All links on HOME page that are for future development will be placeholder links
